This directory contains scripts and commands used for Geosiphon Pyriformis genome analysis

Speedgenomeplots- R scripts used for generating the plots for paper

assemble.sh - used for the Masurca assembly program

commands.sh - includes functional, phylogeny analysis commands and parameters

concoct.sh - contains the binning and separating the clusters of metagenome assembly.

geosiphon_config - config file used for Masurca assembly

msa.sh - multiple sequence commandline for mafft
options.txt - option file used for Fastortho analysis

ReadMe.txt - file your reading.

SNP_calling.sh - parameters used for calling SNPs

SNP_plot_all.R - plot used for generating allele frequency plot

Transposon_bubbleplot.R - script used for generating bubbleplot for transposon classification
